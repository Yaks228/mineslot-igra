## Project Summary
A high-converting landing page for the gambling game "MineSlot", optimized for SEO and modern casino aesthetics. Features a video background and clear CTA buttons for website, Telegram, and VPN access.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (Icons)
- SEO: Metadata API, JSON-LD, Sitemap, Robots.txt

## Architecture
- `src/app`: Application routes and global styles
- `public/`: Static assets (video, favicon.svg, logo, sitemap, robots.txt)
- `index.html`: Standalone static landing page for ultra-fast loading and SEO verification.
- `tailwind.config.ts`: Custom theme for casino aesthetic (dark mode, neon accents)

## User Preferences
- SEO keywords: Mine slot, mine slots, игра, играть, майн слот, майнслот, регистрация, казино, лакибир, лаки бир, играть в телеграм, лаки бир телеграм бот, lucky bear mine slots.

- Modern casino style with dark theme and glowing elements.

## Project Guidelines
- Follow Next.js App Router conventions.
- Use functional components with TypeScript.
- Maintain high SEO standards for Google and Yandex indexing.
